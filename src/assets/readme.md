# 흑백개발자: 코딩 계급 전쟁 (Black & White Developer)

이 프로젝트는 개발자의 코드를 입력받아 "안성재 셰프" 스타일로 재미있게 평가해주는 웹 애플리케이션입니다.

## ✨ 주요 변경 사항 (Result Animation Update)

마지막 결과 화면(`RESULT` State)의 애니메이션과 레이아웃이 대폭 수정되었습니다.

### 1. 분할 레이아웃 (Split Layout)
* 기존의 수직 나열 방식에서 벗어나, 화면을 좌우로 나누어 정보를 배치했습니다.
    * **좌측 (Left):** 결과 이미지 + 등급 (Tier)
    * **우측 (Right):** 심사평 (Verdict) + 다시하기 버튼
* **반응형 지원:** 모바일 화면에서는 자동으로 상하 배치로 전환되어 잘림 없이 표시됩니다.

### 2. 애니메이션 효과 (Framer Motion)
* **좌측 슬라이드 (Slide In):** * 결과 이미지와 등급은 화면 왼쪽 밖에서 안쪽으로 부드럽게 미끄러져 들어옵니다. (`x: -100` -> `0`)
* **우측 페이드인 (Fade In):**
    * 심사평은 좌측 이미지가 자리를 잡은 후, **1초 뒤에** 은은하게 나타납니다. (`delay: 1.0`, `duration: 1.0`)

## 🛠️ 사용 방법

1.  **설치:** `npm install`
2.  **실행:** `npm run dev`
3.  **파일 위치:**
    * `src/App.tsx`: 메인 로직 및 애니메이션 코드가 포함되어 있습니다.
    * `public/images/`: 결과 등급별 이미지 (`result_lv5.png` 등)를 넣어주세요.
    * `public/tasting.gif`: 로딩 화면용 움짤 파일입니다.
    * `public/tasting.mp3`: 로딩 중 재생될 사운드 파일입니다.
